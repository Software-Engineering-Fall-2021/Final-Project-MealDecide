Application Overview

This application will allow the user to take a survey/quiz in order to tell the application what they want to eat or do not want to eat. The user will enter their zip code, tendencies, price range, and answer a set of questions. The application will then return 3-5 restraunts in their area that fit within their criteria. Along with that, the user will be able to choose between wanting fast food, chains restraunts, cheap, sit down, carry out, etc. The user will also have the option to get recipes based on what they have available to them. These questions will revolve around what kind of kitchen the user has and how much time they are willing to spend.
